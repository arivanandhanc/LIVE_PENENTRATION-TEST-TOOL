How it works

Four steps to a full assessment

1 · Set your target & consent

Enter a URL and confirm you're authorised to test it. SSRF guards block private/internal hosts on the hosted service.

2 · Pick a profile

Passive, Standard, or Aggressive — controlling crawl depth and how intrusive the active tests get.

3 · Watch it work, live

Findings and the activity log stream in real time over Server-Sent Events as each module runs.

4 · Export the report

Get a risk-scored, OWASP-mapped report with remediation guidance — one click to PDF.

Built responsibly

Powerful, but principled

This is a defensive security tool for authorised testing. Every scan requires an explicit authorisation confirmation, active tests use safe marker-based detection, and the "DoS resilience" check is a capped, non-destructive probe — never a flood or denial-of-service attack.

Consent gate SSRF protection Non-destructive Rate-limit safe Open source
🛡️

Authorised testing only

Only scan systems you own or have explicit written permission to test. The tool identifies itself in its User-Agent and refuses to scan loopback/private infrastructure on the public host. See the Acceptable Use policy.
